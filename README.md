# Lime

Lime은 [씨미(Cime)](https://ci.me/) 에 여러 기능을 추가하는 브라우저 확장 프로그램입니다.

- 녹화 버튼
- 스크린샷 버튼
- PIP 버튼

확장 팝업 페이지에서 각 기능의 옵션을 설정할 수 있습니다.

## Technologies

- WXT
- React
- TypeScript
- Tailwind CSS

## Install

[Chrome Web Store에서 다운로드하기](https://chromewebstore.google.com/detail/lime-%EC%94%A8%EB%AF%B8-%EB%8F%84%EC%9A%B0%EB%AF%B8/khonbdjmfnmojegcldciagnjcabljcli?authuser=0&hl=ko)

or

```bash
git clone https://github.com/khk4912/Lime
cd Lime
pnpm i
pnpm zip
```

1. 브라우저의 확장 프로그램 페이지 열기
2. 개발자 모드 활성화
3. `dist/` 폴더 로드

## Disclaimer

Lime은 씨미와 관련이 없는 개인 프로젝트입니다.
